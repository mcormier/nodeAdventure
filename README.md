
# nodeAdventure

A text base adventure game built with node.js that deploys to OpenShift

Inspired by DarkSecretSoftware
https://github.com/SandyWalsh/DarkSecretSoftware

Essentially a port to node.js using websockets for communication.


## Reference

The OpenShift `nodejs` cartridge documentation can be found at:

http://openshift.github.io/documentation/oo_cartridge_guide.html#nodejs
